# Notes App

Simple Notes services that allows user to take notes and also user can view, edit and delete his notes at any time

### Features
- Add Note
- Get All Notes
- Get One Note
- Edit Note
- Delete Note

### Technologies & Tools
- JavaScript
- Node.JS
- Express.JS
- MongoDB
